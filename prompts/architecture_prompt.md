# Architecture Planning Prompt

## Purpose
This prompt is used for architecture planning sessions and design reviews.
It helps AI agents generate and evaluate system architecture proposals.

## Template
- Project name: LearnWithNura
- Context: Visual Education OS, slide generation, modular plugins, learning pipelines.
- Current state: Placeholder for existing architecture summary.
- Goals: Placeholder for short-term and long-term architecture goals.
- Constraints: Placeholder for compatibility, maintainability, and documentation rules.
- Outputs expected: Placeholder for architecture diagrams, component definitions, and integration plans.

## Maintenance Notes
- Purpose: Standardize architecture planning conversations.
- Intended contents: Project context, goals, constraints, and deliverables.
- How it should be maintained: Keep aligned with `architecture.md` and `roadmap.md`.
- Relationships: Used alongside `ai_context.md`, `dev_rules.md`, and `feature_prompt.md`.
